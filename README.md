# 109-1 PBC Final Project

## 編號
第38組

## 主題
解析合購版

## 簡述
以requests、BeautifulSoup抓取批踢踢合購版文章，  
再以pandas做資料處理(如擷取年月星期、文章分類、標題分類...等)，  
接著透過plotly、Dash兩套件，製作互動式視覺化圖表，  
最後以Git、Heroku部屬網頁。
